u2pa bdump process FORMAT NUMBER_OF_OUTPUTS NUMBER_OF_INPUTS FILENAME

Processes a binary dump file. Possible formats are:

	ceqn = generates equations for use with CUPL
	ctt  = generates a truth table for use with CUPL
	tt   = generates a truth table in a human readable format
